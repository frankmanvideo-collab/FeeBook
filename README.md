# FeeBook 📚

**Simple fee tracker for Indian tuition teachers**

Built by an 11th grade student in India.

## The Problem
Most Indian tuition teachers track fees in a physical diary.
Apps like Teachmint only have Paid/Unpaid — no partial payment support.
When a teacher collects Rs 1000 cash, the app sends embarrassing
automated reminders to parents because it still shows Unpaid.

## The Solution
- Partial cash payment tracking
- Zero automated reminders — teacher controls everything
- WhatsApp-native reminder workflow
- Works on any phone browser, no download needed
- Free for 15 students

## Tech Stack
- HTML + Tailwind CSS + Vanilla JS
- Supabase (auth + database + RLS)
- Vercel (hosting + serverless API)
- PWA (installable on home screen)
